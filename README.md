\# FlowChain

FlowChain is a blockchain-based trust and reputation management framework for Software Defined Networks (SDN).

\## Features

\- Node Registration

\- Reputation Management

\- Heartbeat Monitoring

\- CPU Monitoring

\- Memory Monitoring

\- Blockchain Audit Logs

\- Real-Time Dashboard

\## Tech Stack

\- Solidity

\- Hardhat

\- Ethereum

\- Express.js

\- Ethers.js

\- HTML/CSS/JavaScript

\- Python

\- Mininet

\## Running Backend

```bash

npm install

npm start


Running Hardhat
npx hardhat node

```
